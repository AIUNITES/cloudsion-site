What Is It

An AI That Publishes Your Website For You

Built and proven on the AIUNITES network of 21 live sites. Every piece runs in production today.

🧠

Claude AI — The Brain

Claude reads your files, understands your site structure, writes and edits HTML/CSS/JS, and coordinates every publishing action without being prompted for each step.

📁

MCP Filesystem — The Hands

Model Context Protocol gives Claude direct read/write access to your local file system. It edits real files on your machine, not a sandboxed copy.

⏰

Silent Auto-Publisher — The Engine

A compiled launcher (no flash, no window) triggers a PowerShell script via Windows Task Scheduler. It commits and pushes every changed repo to GitHub automatically.

🌐

GitHub Pages — The Host

Free, reliable static hosting. Every push goes live within seconds. Custom domains, HTTPS, and global CDN included at no cost.

📊

Analytics Pipeline — The Eyes

Google Apps Script exports feedback and form data to Drive daily. A scheduled task downloads the summary locally. GA4 tracks traffic across all sites from one property.

🔗

Webring — The Network

All sites link to each other through a shared navigation bar, creating internal link equity and a unified brand presence across the entire network.

How It Works

From Prompt to Published in Seconds

The same workflow whether on-prem or cloud — Claude does the heavy lifting.

🏠 On-Premises Flow

1

You Chat with Claude

Open Claude Desktop. Describe what you want — new page, copy edit, layout change, bug fix.

2

Claude Edits Files

Via MCP filesystem, Claude reads your HTML/CSS/JS and writes changes directly to disk.

3

Scheduler Fires

Every 10 minutes, launcher.exe silently triggers auto-publish.ps1 — no window, no flash.

4

Repos Push to GitHub

All changed repos get committed with a timestamp and pushed. GitHub Pages deploys instantly.

5

Sites Go Live

Your changes are live on the web within seconds of the push. Zero manual steps.

☁️ Cloud Flow

1

Log In to Dashboard

Open your Cloudsion workspace in any browser. No installs, no local setup required.

2

Chat with Your AI Agent

Your dedicated Claude instance has access to all your site files in the cloud filesystem.

3

Auto-Publish Runs

A managed cloud cron job (not Task Scheduler) commits and pushes on your schedule.

4

Team Reviews

Teammates can review changes in the dashboard, preview before publish, or approve via Slack.

5

Analytics Roll Up

Cross-site analytics aggregate automatically. See traffic, feedback, and performance in one view.
